# Skrypty

## `scripts/ad`

Skrypty PowerShell związane z Active Directory:

- instalacja AD DS,
- tworzenie OU,
- konta usługowe,
- grupy,
- użytkownicy testowi,
- dołączanie komputerów do domeny.

## `scripts/server`

Skrypty konfiguracji serwerów `SQL01`, `APP01`, `CLIENT01`.

## `scripts/sql`

Skrypty pomocnicze uruchamiane z poziomu PowerShell dla SQL Server.
